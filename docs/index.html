Features
Everything you need,
nothing you don't.

A focused CLI that covers the full Bitbucket Cloud workflow.

Secure Authentication

App Password and OAuth 2.0 support with automatic token refresh. Credentials stored securely via XDG conventions.

Pull Request Workflow

Create, review, approve, merge, and comment on PRs. Add inline comments on specific files and lines. View diffs and activity logs directly in your terminal.

Pipeline Management

Trigger, stop, and monitor CI/CD pipelines. Stream build logs and inspect individual pipeline steps.

Repository Control

List, create, fork, and delete repositories. Browse commits and view diffs across branches.

Flexible Output

Human-readable tables by default, JSON output for scripting. Pipe results into jq, awk, or any tool you like.

Cross-Platform

Pre-built binaries for macOS, Linux, and Windows. Install via Homebrew, go install, or download directly.
